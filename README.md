# CodeAlpha Chatbot Project

## 📌 Overview
This project is a **basic rule-based chatbot** developed as part of the CodeAlpha Python Programming Internship (Task 4).  
The chatbot interacts with users through predefined responses to simple inputs such as greetings, questions, and farewells.

---

🎯 Features
- Responds to basic inputs like:
  - "hello", "hi"
  - "how are you"
  - "bye"
  - "what is your name"
- Provides predefined replies using if-elif conditions.
- Runs in the console with simple input/output.
- Easy to extend with more rules.

---

🛠️ Technologies Used
- Python 3
- Key concepts:
  - functions
  - loops
  - if-elif-else
  - input/output

---

🚀 How to Run
1. Clone this repository:
   `bash
   git clone https://github.com/teja-sri21/CodeAlpha-chatbot.git
   `
2. Navigate into the project folder:
   `bash
   cd CodeAlpha_Chatbot
   `
3. Run the chatbot:
   `bash
   python chatbot.py
   `
